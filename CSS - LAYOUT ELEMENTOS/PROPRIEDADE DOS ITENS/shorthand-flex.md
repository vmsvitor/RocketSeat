# Flex

- shorthand
- flex-grow | flex-shrink | flex-basis
- podem ter 1, 2 ou 3 valores;

<div class="box">
    <div>A</div>
    <div>B</div>
    <div>C</div>
    <div>D</div>
</div>

.box {
    display: flex;
    border: 1px dashed red;
    
}

.box div {
    border: 1px solid;
}   flex: 1;

// quando o valor tem como unidade de medida, o css
entende que se refere ao css, mesmo que esteja na segunda posição.
