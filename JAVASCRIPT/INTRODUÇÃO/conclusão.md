# Tipos de dados

 * Primitive / primitive value
 * Structural
 * Structural Primitive

 # Primitivos - n é um objeto e o valor é imutável

 * Number
 * Boolean
 * undefined
 * Symbol
 * BigInt

 ## Estruturais

* Object

    * Map

    * Date
    * ...

* Function
