# line-height

* espaços entre linhas
* pode ser com unidades ou sem unidades de medida
* comuns: 1.5 ou 2

```css
p {
  line-height: 1.6;
}
```

# text-transform

* transformação do texto

```css
p {
  text-transform: uppercase; /* capitalize (letras iniciais em maiúscula) | lowercase | none*/
}
```
